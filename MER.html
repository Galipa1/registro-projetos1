Relacionamentos e Cardinalidades

Entidade Origem Cardinalidade Entidade Destino Descrição
PROJETISTA 1 : N PROJETO Um projetista pode ser responsável por vários projetos. Um projeto tem exatamente um projetista.
CLIENTE 1 : N PROJETO Um cliente pode solicitar vários projetos. Um projeto pertence a exatamente um cliente.
